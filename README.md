# Elder Med Reminder App

## Problem
Elderly people often forget to take medicines on time.

## Solution
This system helps manage medicine reminders using REST APIs.

## Features
- Add medicines
- View medicines
- Update medicines
- Delete medicines

## Technologies
- Node.js
- Express.js
- MongoDB
- Postman
- GitHub

## API Endpoints

### GET
/api/medicines

### POST
/api/medicines

### PUT
/api/medicines/:id

### DELETE
/api/medicines/:id

## Run Project

npm install

node server.js
